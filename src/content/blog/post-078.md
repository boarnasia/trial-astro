pubDate: 'Jan 29 2022'
---

Vite has become the dominant build tool for modern frontend development. Its development server uses native ES Modules in the browser, eliminating the need to bundle code during development and making the dev server start near-instantaneously.

Vite's production build uses Rollup under the hood, producing highly optimized output with automatic code splitting and tree shaking. The configuration API is simple and well-documented, and the plugin ecosystem covers almost every use case.

Migrating from webpack to Vite is usually straightforward for projects that use standard patterns. The investment pays off immediately in development speed — developers who have made the switch rarely want to go back.
