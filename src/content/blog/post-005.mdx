title: 'Building Your First Astro Site'
description: 'A step-by-step guide to building your first website with Astro, the modern static site framework.'
pubDate: 'May 18 2020'
---

Astro is a modern web framework built around the concept of shipping less JavaScript. Unlike traditional JavaScript frameworks, Astro renders components to static HTML at build time by default, only hydrating interactive components on the client when necessary.

Creating your first Astro project takes just a few commands. The file-based routing system maps .astro files in src/pages/ directly to URL paths, making it intuitive to structure your site. Astro components use a syntax similar to JSX with a frontmatter section for server-side logic.

One of Astro's best features is its compatibility with multiple component frameworks. You can mix React, Vue, Svelte, and Solid components in the same project, using each where it makes the most sense.
