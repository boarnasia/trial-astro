description: 'Learn Node.js fundamentals that every frontend developer should know, including the file system, events, and streams.'
pubDate: 'Apr 27 2023'
---

Node.js brings JavaScript to the server, but many of its core APIs are also useful for frontend tooling, build scripts, and CLI tools. Understanding the basics opens up a new set of possibilities for JavaScript developers.

The file system module — fs — lets you read and write files, which is essential for build scripts and code generators. The path module handles cross-platform path manipulation. Together, they form the foundation of most Node.js tooling.

The http and https modules let you make network requests without a browser. Combined with the streams API — which handles large amounts of data without loading everything into memory — they are powerful tools for data processing and server-side APIs.
