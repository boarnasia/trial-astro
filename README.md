# Trial Astro

Astro ベースのブログプロジェクトです。UI は React、スタイルは SCSS、ヘッドレス UI として Radix UI を利用しています。

## Docs

詳細は以下を参照してください。

- [docs/feontend-design.md](docs/feontend-design.md)
- [docs/onboarding.md](docs/onboarding.md)
- [docs/ai-guidelines.md](docs/ai-guidelines.md)

## Dev URLs

- Astro: http://localhost:4321/
- Storybook: http://localhost:6006/

## Commands

| Command | Action |
| :-- | :-- |
| `bun install` | Install dependencies |
| `bun run build` | Build the project |
| `bun run storybook` | Start Storybook |
| `bun run generate:posts` | Generate sample blog posts |

## Notes

- サーバーの起動と停止は人間が管理します
- 補助スクリプトは [scripts](scripts) 配下にあります
- 進行中タスクは [tasks](tasks) 配下で管理します
